Demonstrate how slots work in components.

www/components/include-slot.html

<div>
  <p>
    This is a slot include.
  </p>

  <slot name='test1'>This is the <strong>default</strong> slot.</slot>
</div>

Default slot

If no slot is provided, the default content is rendered. 

<dj-include template='www/components/include-slot.html' />

Overridden slot

The slot content is replaced by the provided content. 

<dj-include template='www/components/include-slot.html'>
  <span slot="test1">This is the <strong>new</strong> slot.</span>
</dj-include>
This is the new slot.

Missing slot

If a slot is provided that doesn't exist in the component, it is ignored, and the default content for the existing slot is rendered. 

<dj-include template='www/components/include-slot.html'>
  <span slot="test2">This is the new slot.</span>
</dj-include>
This is the new slot.
